# 💡 Infusion Creative Studio

Landing Page desenvolvida para um estúdio criativo digital, com foco em estratégia de marca, web design, desenvolvimento front-end, responsividade e boas práticas de desenvolvimento.

![Preview do Infusion Creative Studio](img/preview.png)

O projeto foi criado inicialmente durante meus estudos de HTML, CSS e JavaScript e posteriormente refatorado, recebendo melhorias de estrutura, conteúdo, semântica, acessibilidade, responsividade, identidade visual, interatividade e organização do código.

## ✨ Funcionalidades

- 🖥️ Página inicial com Hero
- 🧭 Menu de navegação responsivo
- 📱 Menu mobile interativo
- 🎯 Navegação suave entre as seções
- 🎨 Apresentação dos serviços
- 🏢 Seção institucional sobre o estúdio
- 📋 Processo de desenvolvimento organizado em etapas
- 🖼️ Galeria de projetos recentes
- 💻 Carrossel de interfaces
- 💬 Carrossel de depoimentos
- ⏱️ Troca automática dos slides
- ⌨️ Navegação dos carrosséis pelo teclado
- ⏸️ Pausa automática ao interagir com o carrossel
- 📩 Formulário de contato demonstrativo
- 🌐 Links para redes profissionais
- 📱 Layout totalmente responsivo
- ♿ Estrutura HTML semântica
- 🔍 SEO básico
- 🎨 Efeitos de hover e transições
- 🧩 Componentes reutilizáveis
- 🎞️ Suporte à redução de movimento

O formulário de contato é demonstrativo e ainda não possui integração com banco de dados ou serviço de envio.
